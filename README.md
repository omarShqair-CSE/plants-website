# 🌿 Ants Website - Green Your Home

Ants Website is a modern, responsive plant shop website built with **HTML**, **TailwindCSS**, and **JavaScript**. It provides a clean and user-friendly interface for browsing and exploring various indoor plants.

---

## 🚀 Features
- 🏠 **Responsive Design:** Optimized for desktop and mobile devices  
- 🌱 **Plant Showcase:** Display indoor plants with images, descriptions, and prices  
- 🛒 **Shopping Cart Simulation:** Add products to the cart (frontend only)  
- 💬 **Customer Reviews:** Showcase user feedback and ratings  
- 🎨 **Modern UI:** Styled with TailwindCSS for a fresh and natural look  
- ⚡ **Lightweight:** Pure frontend, fast loading without backend  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, JavaScript  
- **Styling:** TailwindCSS  
- **Icons:** RemixIcon  
- **Version Control:** Git & GitHub
